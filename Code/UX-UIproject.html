Proyecto UX-UI

Con un grupo de compañeros hicimos un proyecto de UX-UI enfocado en una aplicacion de futbol fantasia.

Este proyecto abarca la implementación de una solución innovadora para resolver un problema en el deporte del futbol puertorriqueño
Se nos dio la tarea de crear, no solo una solucion, sino un rpoyecto completo utilizando datos aprendidos en UX-UI. Utilizamos el futbol como modelo para crear una app fantasia sobre el deporte y asi crear auge en dicha actividad.
